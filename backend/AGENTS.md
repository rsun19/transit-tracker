## Contributing guide

#### Note: Please run make commands in a `Git Bash` terminal if on Windows.

- If dependencies are not installed, please run `make install-all` in the root.
- To run the repository, please run `make run-backend` in the root.
- To test the repository, please run `make test` in the root.

## Development rules

- Please add all applicable unit tests after contributing code. Please make sure that all code is bug free and we achieve complete testing coverage.
